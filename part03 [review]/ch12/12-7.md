# [치킨 배달](https://www.acmicpc.net/problem/15686)

### 1차시도(성공)

```python
import heapq
from itertools import combinations


def get_dist(a, b):
    return abs(a[0] - b[0]) + abs(a[1] - b[1])


n, m = map(int, input().split())
graph = []
chicken = []
house = []
result = 1e9

for _ in range(n):
    graph.append(list(map(int, input().split())))

for x in range(n):
    for y in range(n):
        if graph[x][y] == 1:
            house.append([x, y])
        elif graph[x][y] == 2:
            graph[x][y] = 0
            chicken.append([x, y])

combination = list(combinations(chicken, m))

for combi in combination:
    total_dist = 0

    for hone in house:
        heap = []
        for chick in combi:
            heapq.heappush(heap, get_dist(chick, hone))

        total_dist += heap[0]

    if total_dist < result:
        result = total_dist

print(result)
```

#### 설명

- 처음에 문제 이해를 제대로 못했다
    1. m개의 치킨집을 고른다 -> combinations함수로 m개의 치킨집을 고를 조합을 구한다.
    1. 고른 치킨집들만 가지고 각 집에서 치킨거리를 구한다
    1. 모든 집의 치킨 거리를 합한 `도시의 치킨거리`를 구한다
    1. `도시의 치킨 거리`의 최솟값을 출력한다
